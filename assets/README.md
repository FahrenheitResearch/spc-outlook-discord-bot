SPC PTS Boundary Assets
=======================

`conus_marine_bnds.txt` is a CONUS/marine boundary used to close raw SPC PTS
outlook contours that terminate on the forecast boundary. It is vendored from
the Iowa Environmental Mesonet pyIEM project:

https://github.com/akrherz/pyIEM

pyIEM is MIT licensed. See the upstream repository for the full license text.
