# Proof Bundle

This directory is a static proof capture from a live dry run of the bot.

It shows the four Discord messages the bot would post:

- Day 1: 4 official SPC map images
- Day 2: 4 official SPC map images
- Day 3: 2 official SPC map images
- Day 4-8: 6 official SPC map images

The HTML page references only local files under `assets/`, and each image has its source URL, byte count, and SHA-256 hash listed on the page.
