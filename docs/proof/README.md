# SPC Outlook Bot Proof Bundles

Generated: 2026-06-13 18:17 UTC

This directory is a static proof page for the fast custom SPC outlook renderer. It contains 4 Discord-style message bundles and 16 PNG maps generated from official SPC geometry products.

Day 1-3 maps are backed by SPC GeoJSON when available. Day 4-8 maps currently use SPC PTS geometry because the SPC page does not expose a matching GeoJSON bundle. Hazard-map legends and overlays use SPC-style Conditional Intensity Group stroke patterns while preserving the warning that these are unofficial fast previews.

Open `index.html` or publish this directory through GitHub Pages.
