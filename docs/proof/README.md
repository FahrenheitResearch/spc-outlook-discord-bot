# Fast Severe Outlook Proof Bundles

Generated: 2026-06-13 18:37 UTC

This directory is a static proof page for the fast custom severe-weather outlook renderer. It contains 4 Discord-style message bundles and 16 PNG maps generated from official NOAA/NWS Storm Prediction Center geometry products.

Day 1-3 maps are backed by SPC GeoJSON when available. Day 4-8 maps currently use SPC PTS geometry because the SPC page does not expose a matching GeoJSON bundle. Hazard-map legends and overlays use SPC-style Conditional Intensity Group stroke patterns while preserving the warning that these are unofficial fast previews.

The generated maps use text attribution only. They intentionally do not include NOAA/NWS/SPC logos or emblems and should not be presented as official SPC/NWS graphics.
